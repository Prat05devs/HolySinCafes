# Holy Sin Cafes – Dehradun’s Exclusive Rooftop Community

This repo contains everything you need to run and deploy the Holy Sin Cafes website locally.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install` or `yarn`
2. Run the app:
   `npm run dev` or `yarn dev`

## Features
- Modern React + Vite setup
- Mobile-first, fully responsive design
- SEO optimized (Open Graph, Schema, meta tags)
- Invite-only registration portal
- Custom cursor and dark mode visuals

## Favicon & Social Sharing
Add your image as `public/holySin.jpg` for favicon and link previews.
